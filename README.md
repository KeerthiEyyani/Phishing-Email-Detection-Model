# Phishing Email Detection Model

This project uses Machine Learning to detect phishing emails.

## Features

- Detects phishing and safe emails
- Uses Scikit-learn
- Displays accuracy
- Shows confusion matrix
- Saves trained model

## Technologies Used

- Python
- Pandas
- Scikit-learn

## Files

- phishing_detector.py
- emails.csv
- model.pkl
